# Space Invaders

The classic Space Invaders game,  
written in Java using OOP principles.

![alt text](./readme_files/gameplay.gif 'Gameplay')

## Installation

1. Clone the repo:  
   `$ git clone https://github.com/sagyas/Space-Invaders.git`
2. Compile:  
   `$ make compile`

## Usage

Run:  
`$ make run`
