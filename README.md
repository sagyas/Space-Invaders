# Space Invaders

The classic Space Invaders game,  
developed with Java using OOP principles.

![alt text](./assets/gameplay.gif 'Gameplay')

## Installation

1. Clone the repo:  
   `$ git clone https://github.com/sagyas/Space-Invaders.git`
2. Compile:  
   `$ make compile`

## Usage

Run: `$ make run`
